# 🏥 ระบบรายงานยอดผู้ป่วยออนไลน์ (Online Census System)

ระบบ Web Application สำหรับบันทึกยอดผู้ป่วยประจำวัน (Census) และภาระงาน (Productivity) ของบุคลากรทางการพยาบาล พัฒนาขึ้นเพื่อเพิ่มประสิทธิภาพในการจัดเก็บและวิเคราะห์ข้อมูลแบบ Real-time รองรับการทำงานทั้งบน Desktop, Tablet และ Smartphone 

**พัฒนาโดย:** พว.วิษณุกรณ์ โอยา (โรงพยาบาลชลบุรี)

---

## ✨ ฟีเจอร์หลัก (Key Features)

* **☁️ Cloud Sync & Local Storage:** ข้อมูลซิงค์กับ Google Sheets อัตโนมัติ (เป็น Database หลังบ้าน) และมีระบบ Local Storage ช่วยให้โหลดข้อมูลได้เร็ว
* **🔐 ระบบจัดการสิทธิ์ (Role-based Access):**
  * **พยาบาลประจำตึก (Ward):** เห็นและกรอกข้อมูลได้เฉพาะหน่วยงานของตนเอง ป้องกันการคีย์ข้อมูลผิดตึก
  * **ผู้ดูแลระบบ (Admin):** มองเห็นภาพรวมทั้งหมด และมีหน้า Dashboard พิเศษสำหรับสรุปข้อมูล
* **📊 Admin Dashboard:** แผงควบคุมสำหรับแอดมิน สามารถกรองข้อมูลดูรายงานย้อนหลังได้ตาม "ช่วงวันที่", "กลุ่มงาน" และ "หน่วยงาน" พร้อมสรุปยอดสถิติแบบ Real-time
* **🧮 Auto-Calculation:** ระบบคำนวณยอดผู้ป่วยคงเหลือให้อัตโนมัติ (ยกมา + รับใหม่ + ย้ายเข้า - จำหน่าย - ย้ายออก - เสียชีวิต)
* **🛏️ Fixed Bed Capacity:** ล็อคจำนวนเตียงตามบริบทจริงของแต่ละหอผู้ป่วยอัตโนมัติ ลด Human Error
* **📱 Responsive Design:** จัดเรียงหน้าจอให้กดง่ายและสวยงามพอดีกับทุกอุปกรณ์ (แบ่ง 6 ช่องสำหรับจอใหญ่ และ 3 ช่อง 2 บรรทัดสำหรับมือถือ)
* **📑 Pagination:** ระบบแบ่งหน้า (หน้าละ 20 รายการ) เพื่อให้หน้าเว็บโหลดไวและดูข้อมูลย้อนหลังได้เป็นระเบียบ

---

## 🛠️ โครงสร้างเทคโนโลยี (Tech Stack)

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Backend / Database:** Google Apps Script (GAS), Google Sheets API (no-cors mode)
* **Hosting:** สามารถรันผ่าน Browser ได้โดยตรง หรือ Host บน GitHub Pages / Cloudflare

## 🚀 การติดตั้งและใช้งาน (Setup Guide)

### 1. การตั้งค่าฝั่งฐานข้อมูล (Google Sheets)
1. สร้างไฟล์ Google Sheets ใหม่ ตั้งชื่อ Sheet ด้านล่างว่า `Census`
2. กำหนดหัวคอลัมน์ (A-Q) ดังนี้: `วันที่`, `เวร`, `หน่วยงาน`, `นอน`, `รับใหม่`, `จำหน่าย`, `ย้ายเข้า`, `ย้ายออก`, `เสียชีวิต`, `คงเหลือ`, `เตียง`, `HN`, `RN`, `TN`, `PN`, `NA`, `ภาระงาน(Productivity)`
3. ไปที่เมนู **ส่วนขยาย (Extensions) > Apps Script**
4. นำโค้ด Backend (ไฟล์ `doGet` และ `doPost`) ไปวาง
5. กด **การทำให้ใช้งานได้ (Deploy) > การทำให้ใช้งานได้รายการใหม่ (New deployment)**
6. เลือกประเภทเป็น **เว็บแอป (Web App)** ตั้งค่าสิทธิ์ให้เข้าถึงได้ "ทุกคน (Anyone)"
7. คัดลอกลิงก์ Web App URL ที่ได้มา

### 2. การตั้งค่าฝั่งหน้าเว็บ (Frontend)
1. เปิดไฟล์ `script.js`
2. ค้นหาตัวแปร `GOOGLE_SCRIPT_URL`
3. นำลิงก์ Web App URL ที่ได้จากข้อ 1 ไปวางในเครื่องหมายคำพูด `""`
```javascript
const GOOGLE_SCRIPT_URL = "[https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec](https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec)";
